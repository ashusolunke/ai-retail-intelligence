# 🛒 AI Retail Intelligence System

An **AI-powered retail analytics dashboard** that helps shop owners and businesses make better decisions using **Machine Learning, demand forecasting, price optimization, and inventory intelligence**.

Built using **Python, Streamlit, and Random Forest Machine Learning**.

---

# 🚀 Live Demo

👉 https://ai-retail-intelligence.streamlit.app

---

# 📊 Features

### 📈 Retail Business Dashboard

PowerBI-style analytics dashboard showing:

* Total products
* Average sales
* Average profit
* Average product price

---

### 🔮 Demand Prediction

Uses **RandomForest Machine Learning** to predict product demand based on:

* Product price
* Visibility
* Weight
* Profit margin

---

### 📉 Demand Forecast

Visualizes how demand changes with price.

Helps businesses understand **price-demand relationships**.

---

### 💹 Price Intelligence

Compares product prices with:

* Market average
* Competitor prices

Provides **AI suggestions** to adjust pricing.

---

### 📦 Inventory AI

Smart system that predicts when inventory should be reordered.

Helps prevent:

* Stockouts
* Overstocking

---

### 💰 Price Optimizer

AI finds the **best price that maximizes revenue** using demand simulation.

---

### 🧠 Explainable AI

Uses **SHAP explainability** to show which features influence demand predictions.

---

# ⚙️ Installation

Clone the repository:

```
git clone https://github.com/ashusolunke/ai-retail-intelligence.git
```

Move to project directory:

```
cd ai-retail-intelligence
```

Install dependencies:

```
pip install -r requirements.txt
```

Run the app:

```
streamlit run app.py
```
